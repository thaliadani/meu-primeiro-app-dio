# Projeto: Meu Primeiro App DIO com Internacionalização
Este projeto marca minha primeira experiência no desenvolvimento de aplicativos Android durante o bootcamp da DIO. O foco principal foi a implementação da internacionalização (i18n), um recurso essencial para alcançar um público global.
## Funcionalidades Implementadas:
- Suporte a 5 idiomas: Adicionei traduções para espanhol, inglês, francês, alemão e russo, modificando o arquivo strings.xml.
- Adaptação automática: O aplicativo detecta o idioma configurado no sistema operacional do dispositivo e exibe o texto na linguagem correspondente de forma automática.
